# 📌 About Wallet Watcher
**Wallet Watcher is a simple yet effective expense tracker built using HTML, CSS, and JavaScript. It helps users keep track of their daily expenses and income with an intuitive and user-friendly interface.**

# 🔹 Features:
✅ **Add & Delete Transactions** – Manage income and expenses easily.  
📊 **Live Balance Updates** – Shows real-time balance calculations.  
💾 **Local Storage Support** – Your data remains saved even after refreshing.  
🎨 **Dark Theme** – Minimalist, eye-friendly UI.
